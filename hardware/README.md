# T113-S3 IoT Station

## 板卡资源

- 主控：全志T113-Sx
- 外设：1.8寸LCD(st7735r)；wifi+bt(rtl8723bs)；麦克风电路；扬声器功放电路与接口；USB-C OTG；SD卡槽与SD NAND预留焊盘；UART调试串口；三合一拨码按键；NRF24L01(与我自己一个课题有关，平时用不到)

## 资料

- 电路设计以LCEDA专业版半离线项目存档(.eprj)，画的比较随意，但是跑满1.1GHz主频基本没问题（T113有点积热，不加散热片基本也只能跑这个频率了）
- 外壳3D模型
- 原理图PDF

## 装配

PCB双层板1.2mm，嘉立创免费打样即可。1.2mm板厚下，外壳刚好可以把所有东西压在一起装配，螺丝采用M1-M1.5左右的微型自攻螺丝即可，PCB的安装螺丝可以不拧。

## 问题

- wifi蓝牙的陶瓷天线信号受限比较严重，调试尽量用USB RNDIS网络，互联网连接可以用wifi或RNDIS共享网络
- 麦克风收音效果一般，AI对话应用中有效收音距离大概只有半米不到，音量随距离衰减较严重，主要是选型问题
- 装配尺寸卡的比较紧，tf卡座可能会顶到LCD背面，可以适当增大一点点后壳的厚度
- 外壳显示区域的开窗忘记留余量了，现在装配后左边部分会有半列到一列的像素被盖住
