# T113-S3 IoT Station

## 板卡资源

- 主控：全志T113-Sx
- 外设：1.8寸LCD(st7735r)；wifi+bt(rtl8723bs)；麦克风电路；扬声器功放电路与接口；USB-C OTG；SD卡槽与SD NAND预留焊盘；UART调试串口；三合一拨码按键；NRF24L01(与我自己一个课题有关，平时用不到)

## 资料

- 电路设计以LCEDA专业版半离线项目存档(.eprj)
- 外壳3D模型
- 原理图PDF

## 问题

- wifi天线受限比较大，调试尽量用USB RNDIS网络，互联网连接可以用wifi或RNDIS共享网络
- 麦克风收音效果一般，AI对话应用中有效收音距离大概只有半米不到，音量随距离衰减较严重，主要是选型问题
